## 1) Генератор **как функция с** `yield`

Это обычная функция, но вместо `return` она использует `yield`.

**Что это даёт:** Функция не возвращает весь результат сразу, а выдаёт элементы по одному — «лениво».

Пример:
```python
def counter():
    for i in range(3):
        yield i
```

Использование:
```python
for x in counter():
    print(x)
```

Вывод:
```python
0
1
2
```

**Главное отличие от обычной функции:**

- обычная функция возвращает результат один раз → и завершает работу
    
- генератор возвращает значения постепенно → и «помнит» своё состояние между вызовами
    

## 2) Генераторное выражение (generator expression)

Это как list comprehension, но в круглых скобках.

Пример:
```python
gen = (x * 2 for x in range(5))
```

Это **не список**, а генератор — выдаёт элементы по одному.

## ✔ Идеальный ответ на собеседовании

> Генератор — это объект, который выдаёт значения по одному, не храня весь результат в памяти. Его можно создать либо функцией с `yield`, либо генераторным выражением в круглых скобках. Генераторы экономят память и работают лениво.
