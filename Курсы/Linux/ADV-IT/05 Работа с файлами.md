
```bash
cat - вывести всё содержимое файла

more - вывести часть(и дальше читать по строкам)

less - показать содержимое файла во вьювере

touch - создание файла

cp (какой файл куда) - копирование файла

rm - удалить файл

rm * или cp  * - применить ко всем файлам  в директории

mv - перенести или переименовать фаил
```
