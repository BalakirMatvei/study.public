```
git init - создать директорию .git в месте, где мы находимся

git config --global user.name/email - юзер представляется гиту

git add test.txt - добавляет файл в staging

git status - выводит информацию о ветке и коммитах

git log - выводит историю коммитов

git diff - показывает изменения относительно текущего коммита

gitt commit -m 'пояснение' - делает коммит 

git checkout (хэш коммита) - путешествие по версиям

git checkout -       - вернуться в версию из которой пришли

git remote - показывает удалённые репозитории

git remote add origin - создает удалённый репозиторий с именем origin

git push/pull origin master - пуш/пул на/с гитхаб 

git clone (ссылка) - полностью копирует репозиторий с гитхаба
```

.gitignore - файл в котром хранится информация о том, что не надо добавлять в гит(каждый создает сам)
