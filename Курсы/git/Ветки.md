```
git branch - показывает какие есть ветки

git branch second -создает новую ветку с именем second

git checkout (название ветки) - путешествие по веткам

git merge second -m - влить ветку second в master(cоздвет ветвление)

git rebase - слияет ветки, но делает структкру плоской(не создавая доп. коммит)

git checkout -b third - создает и переключается на ветку third

git cherry-pick (хэш коммита) - мержит указанные коммиты(или 1 коммит)(не целую ветку) в ветку master

git revert (хэш коммита) - отменяет последний коммит
```
